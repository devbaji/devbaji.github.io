Vue3 Google Login

Lightweight, production-ready plugin for Google Identity Services. One-tap login, OAuth2 flows, and full TypeScript support out of the box.

Google Sign-In Button

Render Google's official sign-in button with a single component. No design work required — fully styled and compliant out of the box.

One Tap & Auto Login

Enable frictionless sign-in with Google's One Tap prompt. Supports automatic login for returning users with zero interaction required.

Custom Login Button

Bring your own button design. Use googleAuthCodeLogin or googleTokenLogin to trigger OAuth2 flows with complete control over the UI.

TypeScript Support

Comprehensive type definitions for all callbacks, options, and API responses. Designed to work seamlessly with Vue 3 script setup.

Nuxt Ready

First-class Nuxt 3 and Nuxt 4 support via the dedicated nuxt-vue3-google-login module, with auto-imports and SSR-safe initialization.

Lightweight

Only 2.3 KiB minzipped with zero runtime dependencies. The Google SDK is loaded asynchronously and only when needed.
